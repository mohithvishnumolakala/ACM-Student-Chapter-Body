CODE COMPANIION




VVIT ACM Student Chapter organized Code Companion on Friday, September 15th from 9:00 AM. The event was part of the AAVEGA 2023 and was attended by students from all years.

 

➢ The event is restricted in time.



➢ The team size is 2.

 


The event received a great response from students with over 250 participants.





About The Event



Code Companion is a pair coding event the students should code as a team. It is a competitive coding event in which the participating teams are provided with a set of coding questions and the participants will be awarded the scores based on the number of questions they solve and the difficulty of the questions.

 

   

The participants are presented with their set of questions using the HackerEarth platform and they are required to have a HackerEarth account prior to the event. They have 90 minutes to solve the problems. Each question receives a grade based on the complexity of the questions.

 

The team which scores the most will be declared as the winner.





 

The event received a good response from students from all academic years, they were very eager to participate in the events being conducted in AAVEGA. participants included students from all branches and academic years of VVIT.
